# Access Control

Repositorio enfocado en documentar buenas prácticas de control de accesos en sistemas Linux y entornos corporativos, incluyendo validación de usuarios, modelos RBAC y principios de menor privilegio.

## Objetivo

Establecer lineamientos claros y auditables para la gestión de usuarios, permisos y accesos a recursos críticos.

## Alcance

- Gestión de usuarios y grupos
- Principio de menor privilegio
- Modelos de control de acceso (RBAC, DAC, MAC)
- Validación de accesos y auditoría
- Documentación clara y defendible

## Enfoque

- Cambios progresivos y auditables
- Explicaciones técnicas justificadas
- Adaptable a entornos reales

## Estado del proyecto

Documentación inicial en desarrollo.
