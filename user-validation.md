# User Validation

Este documento describe procedimientos para validar usuarios y accesos en sistemas Linux y entornos corporativos.

## Objetivo

Garantizar que solo usuarios autorizados tengan acceso a recursos críticos y que los cambios se auditen correctamente.

## Validación de usuarios

- Revisar usuarios activos periódicamente
- Confirmar pertenencia a grupos correctos
- Validar credenciales seguras (contraseñas o llaves)
- Revisar cuentas compartidas o duplicadas

## Validación de accesos

- Revisar permisos de archivos y directorios críticos
- Validar políticas de sudo y escalamiento
- Registrar cambios en auditoría
- Ejecutar revisiones periódicas de accesos
